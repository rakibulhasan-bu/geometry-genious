What is event bubble?

Event Bubbling is a concept in the DOM. It's means that when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element. If we wanna stop this bubbling that use event.stopPropagation() in the section what we wanna stop.
